# FASHIO — Premium Minimalist E-Commerce Store

FASHIO is a high-end, responsive Single Page Application (SPA) e-commerce storefront built for a universal audience. Featuring a clean, luxury layout inspired by modern design frameworks, it includes state-managed cart drawers, persistent order history tracking timelines, and a fully optimized keyword product search engine.

## 🚀 Key Features

* **Interactive Shop Grid:** Fully functional search engine capable of dynamically filtering across diverse product categories including Hats, Jackets, Sunglasses, Shirts, Blazers, Coats, and Pants.
* **State Management Flow:** Powered by Redux Toolkit to track item selection options, update cart quantities, and calculate exact totals in real-time.
* **Secure Routing Layout:** Integrated with React Router DOM to manage fluid view transitions without forcing manual browser reloads.
* **Full-Stack Database Sync:** Wired with an Axios communication layer to save finalized client order configurations into a permanent cloud MongoDB collection.
* **Account Dashboard UI:** Complete with secure session logout handling, user validation data hooks, and a visual shipping tracking timeline.

## 🛠️ Tech Stack

* **Frontend:** React.js, Redux Toolkit, React Router DOM, Tailwind CSS
* **Backend Communication:** Axios HTTP Client API
* **Database Workspace:** MongoDB (Cloud Atlas Cluster via Mongoose ODM)
